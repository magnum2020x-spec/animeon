# 🎬 AnimeBest - منصة مشاهدة الأنمي العربية

منصة عصرية وفاخرة لمشاهدة الأنمي بتصميم احترافي وتجربة مستخدم فريدة.

## 📁 محتويات المشروع

```
AnimeBest-Complete/
├── index.html              # الصفحة الرئيسية (ملف HTML واحد شامل)
├── standalone.html         # نسخة بديلة مع تصميم إضافي
├── README.md              # هذا الملف
├── assets/
│   ├── css/
│   │   └── style.css      # ملف التنسيق الشامل
│   ├── js/
│   │   └── main.js        # ملف JavaScript الشامل
│   ├── index.html         # صفحة HTML تستخدم الملفات المنفصلة
│   └── README.md          # دليل استخدام Assets
```

## 🛠️ التخصيص

### تغيير الألوان
ابحث عن قسم `:root` في CSS وعدّل المتغيرات:

```css
:root {
    --primary: #a855f7;      /* اللون الأساسي */
    --secondary: #06b6d4;    /* اللون الثانوي */
    /* ... */
}
```

### تغيير النصوص
ابحث عن النصوص العربية في HTML وعدّلها حسب احتياجاتك.

### إضافة صور حقيقية
استبدل روابط الصور البديلة بروابط صورك الخاصة:

```html
<img src="your-image-url.jpg" alt="description">
```

## 🚀 النشر

### النشر على GitHub Pages
```bash
1. أنشئ repository جديد
2. ارفع جميع الملفات
3. اذهب إلى Settings → Pages
4. اختر main branch
5. انسخ الرابط الذي يظهر
```

### النشر على Netlify
```bash
1. اذهب إلى netlify.com
2. اسحب وأفلت مجلد AnimeBest-Complete
3. سيتم النشر تلقائياً
```

### النشر على Vercel
```bash
1. اذهب إلى vercel.com
2. اختر "Import Project"
3. اختر repository الخاص بك
4. انقر "Deploy"
```

## 🐛 استكشاف الأخطاء

### الصفحة لا تظهر بشكل صحيح
- تأكد من فتح الملف في متصفح حديث
- امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
- جرب متصفح آخر

### الصور لا تظهر
- تحقق من اتصالك بالإنترنت
- تأكد من صحة روابط الصور
- استبدل الصور البديلة بصور حقيقية

### الرسوم المتحركة لا تعمل
- تأكد من تفعيل JavaScript في المتصفح
- جرب متصفح آخر
- افتح أدوات المطور (F12) وتحقق من الأخطاء
